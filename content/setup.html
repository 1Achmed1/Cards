Cards Inital Setup

Cards needs to be set up to work properly. Please follow the steps below and we'll make sure you've done everything correctly.

  1. Head over to Parse.com
  2. Click "Sign up" in the top-right corner of the browser window
  3. Sign in with Facebook, Github, or Google. You can also manually enter all of your information. If you do, name your first project what you want the title of your website to be.
  4. At the top bar (should be gray) click "Core".
  5. On the left, click "Config".
  6. Click "Create your first parameter".
  7. Name the parameter "websiteTitle", without the quotes. Capitalization matters, so just copy/paste what I've put in the quotes.
  8. Set the box that says "Value" to the title of your website. This title will show up in the title bar of tabs or the window.
  9. Go ahead and make the following parameters next:
      • Title: "setUpCompleted"
      • Type: Boolean
      • Value: false
      • Title: "contentDir"
      • Type: String
      • Value: Set this to "content" if you don't know what this is.
      • Title: "fileEndings"
      • Type: String
      • Value: Set this to what most of your files will be. Leave as "html" if you don't know what this is.
  10. Once you're done with that, you should have four different parameters. Now click the big "P" in the top-right corner of the screen.
  11. On your project you should see a blue link that says "quickstart guide". Click it.
  12. Click the icon that says "Data".
  13. Click "Web".
  14. Click "Existing Project".
  15. Copy everything from the second code box. It should have random numbers and letters in it.
  16. Open the file content-loader.js from your website's root directory using an application like Notepad or TextEdit. Do not use WordPad, MS Word, Pages, etc.
  17. Paste everything you copied onto the third line, overwritting everything that's there.
