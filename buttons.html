Cards
Buttons

All of the 500 values of the Material Design colors are included in Cards. Buttons take on these colors.

Buttons will take on the color associated in the class as well as the corresponding color. Not specifying a color will result in the use of the header color.

Code Example

You can specify a button's color with it's class attribute.

Example:

<button class="ripple red">This button is red!</button>

Produces

The ripple class is required to make the button have the ripple effect. The color in the class is what color the button should be. Usable colors are specified in Color.
